3 Преобразование типов

важность: 5

Какой результат будет у выражений ниже?

"" + 1 + 0 // "" + "1" + "0" = "10" + Сложение со строкой "" + 1 преобразует 1 к строке: "" + 1 = "1", и в следующем случае "1" + 0 работает то же самое правило.;

"" - 1 + 0 // 0 - 1 + 0 = -1 + Вычитание - (как и большинство математических операторов) работает только с числами, пустая строка "" приводится к 0.;

true + false // 1 + 0 = 1 +;

6 / "3" // 6 / 3 = 2 +;

"2" * "3" // 2 * 3 = 6 +;

4 + 5 + "px" // 4 + 5 = 9, 9 + "px" = "9px" +;

"$" + 4 + 5 // "$" + 4 = "$4", "$4" + 5 = "$45" +;

"4" - 2 // 4 - 2 = 2 +;

"4px" - 2 // NaN +;

" -9 " + 5 // = " -9 5" + Сложение со строкой превращает число 5 в строку и добавляет к строке.;

" -9 " - 5 // -9 - 5 = -14 + Вычитание всегда преобразует к числу, значит строка " -9 " становится числом -9 (пробелы по краям обрезаются).;

null + 1 // 0 + 1 = 1 + null становится 0 после численного преобразования.;

undefined + 1 // NaN + 1 = NaN + undefined становится NaN после численного преобразования.;

" \t \n" - 2 // 0 - 2 = -2 + Пробельные символы, такие как \t и \n, по краям строки игнорируются при преобразовании в число, так что строка " \t \n", аналогично пустой строке, становится 0 после численного преобразования.;
